# 🏸 Badminton Setup

> Ứng dụng Quản lý Giao lưu Cầu lông Chuyên nghiệp

**Số hóa và tự động hóa quy trình tổ chức các buổi đánh cầu lông** — thay thế hình thức bình chọn trên Zalo/Messenger.

---

## ✨ Tính năng

| Tính năng | Mô tả |
|---|---|
| 📋 **Điểm danh** | Xác nhận tham gia, báo vắng, thêm khách mời |
| 🏟️ **Sơ đồ sân** | Kéo thả người chơi vào từng sân. Xem ai đang đấu, ai chờ |
| ⚡ **Ghép cặp** | Tự động ghép cặp cân bằng theo trình độ |
| 💰 **Chi phí** | Chia đều chi phí, QR code thanh toán nhanh |
| 📊 **Lịch sử** | Xem lại toàn bộ lịch sử các buổi giao lưu |

---

## 🚀 Chạy local

```bash
npm install
npm run dev
```

Mở [http://localhost:5173](http://localhost:5173)

## 🏗️ Build

```bash
npm run build
```

---

## 🛠️ Stack

- **Vue 3** (Composition API)
- **Pinia** (state management)
- **Vue Router** (navigation)
- **qrcode** (QR generation)
- **Vite** (build tool)
- **Vanilla CSS** (custom design system)

---

## 🎨 Design

**Sport Brutalism** — Dark background × Acid Lime × Aggressive Orange.  
Typography: Rajdhani (headers) + DM Sans (body).
